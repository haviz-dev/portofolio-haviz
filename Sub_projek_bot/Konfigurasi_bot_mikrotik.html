Tentang Bot MikroTik

Bot Telegram ini menyediakan tiga fitur utama untuk mempermudah pengelolaan perangkat MikroTik langsung dari Telegram. Pertama, Konfigurasi IP memungkinkan pengguna untuk mengatur, mengecek, dan menghapus alamat IP secara langsung melalui perintah chat. Kedua, Manajemen DHCP Server mendukung pengaturan, pemantauan, dan penghapusan konfigurasi DHCP seperti pool, network, gateway, dan interface dengan praktis. Ketiga, Manajemen Firewall memudahkan pengguna dalam menambah, mengecek, dan menghapus rule firewall tanpa perlu mengakses perangkat secara manual. Semua proses dilakukan secara instan dan efisien hanya melalui Telegram. Proyek ini dibangun menggunakan Python dengan integrasi python-telegram-bot dan routeros_api untuk memastikan pengelolaan jaringan yang efisien dan terstruktur serta menggunakan GNS3 untuk melakukan pengetesan secara real-time.

Untuk fitur Manajemen Firewall, bot menggunakan sistem key agar lebih fleksibel. Pengguna cukup mengirimkan key tertentu (seperti chain, action, src-address, dll) melalui Telegram, dan bot akan membaca serta memproses key tersebut sebagai variabel. Setelah itu, bot melakukan parse perintah berdasarkan key yang dimasukkan, mengirimkannya ke perangkat MikroTik, lalu mengembalikan hasilnya ke pengguna secara langsung melalui chat. Dengan sistem ini, pengguna dapat mengatur, mengecek, atau menghapus rule firewall dengan lebih dinamis dan tanpa perlu menyebutkan struktur perintah lengkap.

Fitur Bot Telegram MikroTik

Identitas MikroTik:

Melalui perintah /set_nama, pengguna dapat menyimpan nama atau identitas router agar lebih mudah dikenali, terutama saat mengelola banyak perangkat MikroTik.

Konfigurasi IP:

Bot dapat digunakan untuk mengatur, mengecek, dan menghapus alamat IP pada perangkat MikroTik. Perintah seperti /set_ip, /cek_ip, dan /hapus_ip memungkinkan pengguna melakukan konfigurasi IP langsung dari Telegram.

Manajemen DHCP Server:

Bot menyediakan fitur untuk mengatur, mengecek, dan menghapus konfigurasi DHCP Server pada perangkat MikroTik. Ini mencakup pengelolaan DHCP pool, network, gateway, dan interface, yang bisa diakses melalui /set_dhcp, /cek_dhcp, dan /hapus_dhcp.

NAT Configuration:

Fitur /set_nat digunakan untuk menambahkan rule NAT pada konfigurasi firewall MikroTik, misalnya untuk keperluan koneksi internet NAT masquerade.

Manajemen Firewall:

Melalui /set_filter_firewall, /list_filter_firewall, dan /hapus_filter_firewall, pengguna dapat mengatur dan mengelola filter firewall. Bot mendukung penghapusan berdasarkan sistem key seperti chain, action, comment, src-address, dst., yang membuat pengelolaan firewall menjadi lebih cepat dan efisien.

Koneksi MikroTik:

Melalui perintah /cek_koneksi_MikroTik, pengguna dapat memverifikasi apakah bot berhasil terhubung dengan perangkat MikroTik secara real time.

Keamanan Bot dan Autentikasi:

Bot dilengkapi dengan sistem keamanan berbasis autentikasi pengguna. Hanya pengguna yang telah ditentukan sebelumnya dalam konfigurasi bot (file .env) yang bisa mengakses dan mengendalikan perangkat MikroTik, sehingga mencegah akses tidak sah.

Bantuan Pengguna:

Dengan perintah /help, pengguna dapat melihat semua daftar perintah yang tersedia lengkap dengan penjelasannya, sehingga mempermudah proses pembelajaran dan penggunaan fitur bot.

Projek MikroTik

Otomasi MikroTik via Bot Telegram — GNS3 | Python | JSON

Fitur : /set_ip, /cek_ip, /hapus_ip, /set_dhcp, /cek_dhcp, /hapus_dhcp, /set_firewall, /cek_firewall, /hapus_firewall, dan masih banyak lagi

Stack: Python, Telegram Bot API, RouterOS API

Tahun: 2025

Bot ini dibuat untuk mengotomasi konfigurasi router MikroTik langsung dari Telegram secara efisien. Terintegrasi dengan GNS3 untuk simulasi jaringan.

Topologi Jaringan

Gambar: Topologi Jaringan MikroTik di GNS3

Topologi: Jaringan hanya terdiri dari satu router MikroTik dan beberapa klien (client). Konfigurasi dilakukan secara fleksibel menggunakan Bot Telegram tanpa perlu akses ke laptop atau komputer.

Koneksi Internet: Router MikroTik mendapatkan akses internet dari WiFi melalui port ether1 dan menyalurkan koneksi internet ke klien melalui port ether2.

Konfigurasi Cepat dan Lengkap: Dengan memanfaatkan fitur-fitur yang disediakan Bot Telegram, pengguna dapat mengatur IP, DHCP Server, NAT, Firewall, dan identitas router dengan mudah — cukup dari ponsel. Ini sangat membantu jika tidak tersedia perangkat desktop untuk melakukan pengaturan jaringan.
