Tentang Bot MikroTik Massal

Bot Telegram ini dirancang untuk menyederhanakan proses konfigurasi massal perangkat MikroTik hanya lewat chat. Dengan dukungan file JSON, bot mampu mengatur banyak router sekaligus secara otomatis, serta menyediakan fitur pengecekan IP, serta menghapus IP secara massal. Proyek ini dibangun menggunakan Python dengan integrasi python-telegram-bot, routeros_api, dan json untuk memastikan pengelolaan jaringan yang efisien dan terstruktur serta menggunakan GNS3 untuk melakukan pengetesan secara real-time

Bot Akan terus dikembangkan sehingga menjadi lebih efisien. Seperti fitur DHCP-Server, Firewall, Hospot, dan masih banyak lagi. Dan saya program ini akan menjadikan konfigurasi semakin mudah. Cukup satu klik pada Bot, semua router terkonfigurasi.

Fitur Bot Telegram MikroTik

Set IP Massal:

Bot dapat mengatur alamat IP ke banyak perangkat MikroTik sekaligus hanya dengan satu perintah melalui Telegram. Proses ini dilakukan berdasarkan data konfigurasi dari file JSON yang berisi detail interface, IP, dan jaringan tiap router.

Cek IP Massal:

Bot akan menampilkan daftar alamat IP dari seluruh perangkat yang telah terkoneksi, membantu pengguna memantau konfigurasi IP secara menyeluruh.

Hapus IP Massal:

Bot memungkinkan pengguna untuk menghapus alamat IP yang telah dikonfigurasi pada banyak perangkat MikroTik sekaligus, cukup dengan satu perintah. Proses ini menggunakan data yang ada dalam file JSON untuk memastikan IP yang akan dihapus sesuai dengan konfigurasi yang telah ditetapkan.

Projek MikroTik

Fitur : /set_ip_Massal, /cek_ip_massal, /hapus_ip_semua

Stack: Python, Telegram Bot API, RouterOS API, JSON

Bot ini dibuat untuk mengotomasi konfigurasi banyak router MikroTik langsung dari Telegram secara efisien. Terintegrasi dengan GNS3 untuk simulasi jaringan dan file JSON untuk pengaturan massal.

Topologi Jaringan

Gambar: Topologi Jaringan GNS3 setelah di konfiguasi Massal

Topologi: Jaringan dapat terdiiri 2 hingga lebih banyak router. Konfigurasi dilakukan secara fleksibel dan massal menggunakan Bot Telegram tanpa perlu akses ke laptop atau komputer dengan catatan, daftar konfigurasi harus sudah tersedia di json.

Koneksi Internet: Router MikroTik mendapatkan akses internet dari WiFi melalui port ether1 dan menyalurkan koneksi internet ke klien melalui port ether2.

Konfigurasi Cepat dan Lengkap: Dengan memanfaatkan fitur yang tersedia saat ini di Bot Telegram, pengguna dapat konfigurasi IP secara massal hanya dengan satu kali klik
