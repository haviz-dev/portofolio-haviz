Tentang Proyek UTBK SMKN 1 Mandau (2019)

Pada tahun 2019, saya terlibat dalam proyek infrastruktur jaringan untuk mendukung pelaksanaan UTBK (Ujian Tulis Berbasis Komputer) di SMKN 1 Mandau. Proyek ini bertujuan untuk membangun sistem jaringan LAN yang stabil dan efisien sebagai fondasi utama pelaksanaan UTBK di sekolah.

Saya merancang dan menyimulasikan topologi jaringan menggunakan GNS3, menghubungkan seluruh klien melalui switch, serta mengonfigurasi server MikroTik dengan pengaturan DHCP Server, Firewall, dan Routing. Selain itu, saya juga melakukan instalasi dan konfigurasi perangkat lunak UTBK di tiap perangkat klien agar seluruh sistem siap digunakan.

Hasil akhir dari proyek ini adalah jaringan terstruktur yang mendukung pelaksanaan UTBK secara optimal dan stabil. Pengalaman ini juga menjadi salah satu tonggak awal pemahaman saya tentang manajemen jaringan dalam skala institusi pendidikan.

Kontribusi Proyek

Perancangan Jaringan:

Merancang topologi jaringan LAN menggunakan GNS3 untuk kebutuhan UTBK.

Konfigurasi MikroTik:

Melakukan konfigurasi DHCP Server, Firewall, dan Routing pada MikroTik untuk mendukung distribusi IP otomatis, keamanan jaringan, dan komunikasi antar perangkat.

Instalasi UTBK Software:

Menginstal dan mengonfigurasi perangkat lunak UTBK pada setiap klien untuk mendukung pelaksanaan ujian secara daring.

Projek MikroTik

Topologi Jaringan

Gambar: Topologi Jaringan GNS3 setelah di konfiguasi Massal

Topologi: Jaringan terdiri dari 3 ruangan dengan subnet yang sama. Kepala sekolah menginginkan setiap komputer dapat terkoneksi ke internet dan mengakses aplikasi utbk dengan stabil. kami kemudian merancang jaringan dengan menggunakan GNS3 dengan server utama menggunakan mikrotik. Dan menggunakn NAT agar client dapat terhubung ke firewall dan memblokir koneksi antar subnet.

Koneksi Internet: Router MikroTik mendapatkan akses internet melalui port ether1 dan menyalurkan koneksi internet ke klien melalui port ether2.

Menggunakan Firewall Filter: menghubungkan dengan internet, tetapi melarang koneksi antar jaringan LAN.

Firewall Filter

Fitur: Mengizinkan akses ke internet, tetapi dilarang untuk komunikasi antar jaringan LAN

Stack:


# Allow traffic to internet
add chain=forward out-interface=ether1 action=accept comment="Allow LAN to Internet"
add chain=forward in-interface=ether1 action=accept comment="Allow Internet responses"

# DROP traffic antar LAN
add chain=forward src-address=192.168.100.2-192.168.100.50 dst-address=192.168.100.51-192.168.100.100 action=drop comment="Block LAN1 to LAN2"
add chain=forward src-address=192.168.100.2-192.168.100.50 dst-address=192.168.100.101-192.168.100.150 action=drop comment="Block LAN1 to LAN3"
add chain=forward src-address=192.168.100.51-192.168.100.100 dst-address=192.168.100.2-192.168.100.50 action=drop comment="Block LAN2 to LAN1"
add chain=forward src-address=192.168.100.51-192.168.100.100 dst-address=192.168.100.101-192.168.100.150 action=drop comment="Block LAN2 to LAN3"
add chain=forward src-address=192.168.100.101-192.168.100.150 dst-address=192.168.100.2-192.168.100.50 action=drop comment="Block LAN3 to LAN1"
add chain=forward src-address=192.168.100.101-192.168.100.150 dst-address=192.168.100.51-192.168.100.100 action=drop comment="Block LAN3 to LAN2"

# Allow the rest (optional)
add chain=forward action=accept comment="Allow the rest of forwarding"
